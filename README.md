# study-of-conditional-statement
In this experiment, we studied the concept of conditional statements in Python, which are used to control the flow of program execution based on certain conditions. Conditional statements help the program make decisions and execute different blocks of code depending on whether a condition is true or false.

We first implemented a program to check whether a given number is positive or negative using the if–else statement. Then we modified the program using if–elif–else statements to also detect the case when the number is zero. This helped us understand how multiple conditions can be checked sequentially.

Next, we wrote a program to determine whether a number is even or odd by using the modulus operator (%), which checks the remainder after division by 2.

After that, we implemented programs to find the largest number among two numbers and among three numbers using comparison operators and logical operators such as and. These programs demonstrated how conditional statements can be used for comparisons between multiple values.

We also created a program that accepts marks of five subjects, calculates the average percentage, and assigns a grade according to predefined grading criteria. This program helped us understand the use of multiple elif conditions and how they can be used to categorize values into ranges.

Further, we wrote a program to check whether a given year is a leap year. The program uses the standard leap year rules:

A year is a leap year if it is divisible by 400, or

If it is divisible by 4 but not divisible by 100.

Finally, we implemented programs to validate a given date by checking the number of days in each month, including the special case of February in leap years. We also created a program that prints the incremented date (next day) if the entered date is valid. This required checking the maximum number of days in each month and updating the day, month, and year accordingly when the date changes.

Overall, this experiment helped us understand how decision-making structures are implemented in Python, and how they can be applied to solve practical problems involving comparisons, logical conditions, and validations.

#Conclusion

From this experiment, we successfully learned how to use conditional statements in Python, including if, if–else, and if–elif–else constructs, to control the flow of a program. We applied these concepts to solve different types of problems such as checking number properties, comparing numbers, assigning grades based on average marks, determining leap years, and validating calendar dates. These programs demonstrated how logical conditions and comparison operators are used to implement decision-making in programming. The experiment improved our understanding of problem-solving using conditional logic, which is a fundamental concept in Python programming and software development.
